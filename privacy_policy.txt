4. ХРАНЕНИЕ И ЗАЩИТА ДАННЫХ
================================================================================

4.1. МЕСТНОЕ ХРАНИЛИЩЕ

Некоторые данные хранятся локально на вашем устройстве:
• Токены аутентификации (в защищенном хранилище)
• Настройки языка и темы
• Кэш учебных материалов
• Временные файлы

4.2. СЕРВЕРНОЕ ХРАНИЛИЩЕ

Следующие данные хранятся на наших защищенных серверах:
• Информация профиля пользователя
• Академические данные
• История обучения и тестирования
• История диалогов с AI-ассистентом
• Данные о подписках и покупках

4.3. МЕРЫ БЕЗОПАСНОСТИ

Мы применяем следующие меры безопасности:
• Шифрование данных при передаче (HTTPS)
• Защищенное хранение токенов доступа
• Автоматическое обновление токенов
• Регулярные проверки безопасности
• Ограничение доступа к персональным данным
• Автоматический выход при обнаружении подозрительной активности

4.4. СРОКИ ХРАНЕНИЯ

• Данные активной учетной записи хранятся весь период использования Приложения
• После удаления учетной записи персональные данные удаляются в течение 30 дней
• Некоторые данные могут храниться дольше для соблюдения законодательных требований

5. РАЗРЕШЕНИЯ ПРИЛОЖЕНИЯ
================================================================================

Приложение запрашивает следующие разрешения на вашем устройстве:

5.1. ДОСТУП К МИКРОФОНУ (RECORD_AUDIO)
• Для записи аудио в образовательных целях
• Для использования голосовых функций
• Разрешение используется только с вашего явного согласия

5.2. ИЗМЕНЕНИЕ НАСТРОЕК АУДИО (MODIFY_AUDIO_SETTINGS)
• Для оптимизации воспроизведения учебных материалов
• Для настройки громкости и качества звука

5.3. ДОСТУП К КАМЕРЕ И ГАЛЕРЕЕ
• Для загрузки фотографии профиля
• Для загрузки изображений в маркете (при необходимости)
• Используется только с вашего явного разрешения

5.4. ДОСТУП К ФАЙЛОВОЙ СИСТЕМЕ
• Для сохранения и чтения учебных материалов
• Для кэширования контента
• Для загрузки и хранения файлов

5.5. ДОСТУП К БУФЕРУ ОБМЕНА
• Для копирования текста из учебных материалов
• Для удобства работы с контентом

12. ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ
================================================================================

12.1. COOKIES И ПОДОБНЫЕ ТЕХНОЛОГИИ

Мобильное приложение не использует cookies в традиционном понимании, но 
использует аналогичные технологии:
• AsyncStorage для локального хранения настроек
• Secure Store для безопасного хранения токенов
• Кэширование для улучшения производительности

12.2. ПОДПИСКИ И ПЛАТЕЖИ

При оформлении подписки:
• Мы собираем информацию о типе и статусе подписки
• Платежные данные обрабатываются через защищенные платежные системы
• Мы не храним данные платежных карт
• Информация о транзакциях хранится для учетных целей
• Доступен бесплатный пробный период

12.3. AI-АССИСТЕНТ

При использовании AI-ассистента:
• Мы сохраняем историю диалогов для улучшения опыта
• Данные используются для обучения и улучшения AI-модели
• Вы можете удалить историю диалогов в любое время
• Персональная информация не используется для обучения сторонних AI-моделей

12.4. МНОГОЯЗЫЧНАЯ ПОДДЕРЖКА

Приложение поддерживает следующие языки:
• Английский (English)
• Русский (Русский)
• Таджикский (Тоҷикӣ)
• Казахский (Қазақша)
• Узбекский (O'zbek)
• Киргизский (Кыргызча)

Ваш выбор языка сохраняется локально и используется для персонализации.
